# Power BI Team Performance Analysis

This repository contains a Power BI dashboard for analyzing sales team performance.

## Dashboard Overview

The report tracks key sales performance indicators including:

- Sell Out
- Strike Rate
- Call Completion
- % ASO
- % TLS
- On-route vs off-route sell-out
- Sales representative performance

## Key Insights

- The team has average call completion above 70%, but strike rate is relatively low at around 59.8%.
- Off-route orders contribute a large share of sell-out, suggesting the current route plan may need review.
- Some sales representatives have strong sell-out but low strike rate, showing opportunities to improve call effectiveness.
- Top-performing representatives can be used as best-practice examples for the wider team.
